

## Create Scripts & Guides DOCX Files

Generate 10 professional DOCX documents for real estate agents, saved to `/mnt/documents/` for upload to the Resources tab.

### Documents to Create

1. **Cold Calling Script — Expired Listings** — Opening lines, pain point questions, objection responses, appointment close
2. **Cold Calling Script — FSBOs** — Value proposition, "why use an agent" framework, common objections
3. **Cold Calling Script — Circle Prospecting** — "A home just sold nearby" conversation flow
4. **Buyer Consultation Guide** — Meeting agenda: needs assessment, pre-approval, timeline, agency disclosure
5. **Seller Consultation Guide** — CMA walkthrough, pricing strategy, marketing plan, listing agreement
6. **Objection Handler Playbook** — Top 20 objections with word-for-word responses (commission, timing, exclusivity, etc.)
7. **Open House Conversion Script** — Greeting, qualifying questions, registration pitch, follow-up plan
8. **Sphere of Influence Touchpoint Guide** — Monthly/quarterly contact cadence: calls, texts, pop-bys, events
9. **New Agent 90-Day Action Plan** — Daily/weekly activity targets, prospecting goals, training milestones
10. **Negotiation Tactics Guide** — Multiple offers, inspection responses, appraisal gaps, escalation clauses

### Approach

- Use `docx-js` via a Node.js script to generate each file
- Consistent branding: clean layout, Arial font, professional headers
- US Letter size, proper margins
- Each doc uses tables, bullet lists, and section headers for easy scanning
- All files saved to `/mnt/documents/resources/scripts-and-guides/`
