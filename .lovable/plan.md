

## Add Facebook Pixel to index.html

Add the Meta Pixel script (ID `1730908397470525`) to `index.html`:

1. **`<head>`**: Insert the Facebook Pixel `<script>` block alongside the existing GTM/GA4/Clarity scripts.
2. **`<body>`**: Insert the `<noscript>` fallback image right after the existing GTM noscript iframe.

Single file change, same pattern as the other tracking scripts already installed.
